# 📊 Task 2 – Sales Data Analysis Dashboard (Future Interns)

## 🎯 Project Overview
As part of **Task 2 in the Future Interns Program**, I developed an **interactive Sales Analysis Dashboard** using **Power BI**.  
The objective of this task was to transform raw sales data into meaningful insights through **data cleaning, data modeling, and visualization**.

This project demonstrates how raw datasets can be processed and visualized to support **data-driven decision making**.

---

## 🛠 Tools & Technologies
- 📊 **Power BI** – Dashboard creation and data visualization  
- 📂 **Microsoft Excel** – Data cleaning and preprocessing  
- 🧠 **Data Modeling** – Creating relationships between tables  

---

## 🧹 Data Preparation
Before building the dashboard, the dataset was cleaned using **Microsoft Excel**.

Key steps performed:

- ✔️ Removed duplicate records  
- ✔️ Handled missing values  
- ✔️ Standardized column formats  
- ✔️ Structured the dataset for analysis  

This preprocessing ensured that the data was **accurate and ready for analysis in Power BI**.

---

## 🧠 Data Modeling
After importing the cleaned dataset into **Power BI**, a proper **data model** was created.

This included:

- 🔹 Creating relationships between multiple tables  
- 🔹 Structuring the dataset for better analysis  
- 🔹 Enabling accurate filtering across visualizations  

The data model allows insights to be generated across **products, regions, and sales metrics**.

---

## 📈 Dashboard Features
The dashboard provides interactive insights into sales performance:

- 📊 **Revenue Analysis**
- 📦 **Product Performance**
- 🌍 **Regional Sales Comparison**
- 📅 **Sales Trends Over Time**
- 🎛 **Interactive Filters and Slicers**

These visualizations help transform complex data into **clear and meaningful insights**.
